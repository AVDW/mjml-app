    Mjml app



    The first MJML Email Editor


    This is a local editor for the MJML language. It allows you to create responsive emails with a live preview on both phone and Desktop.

    The app is based on React, Redux and Electron.

    Installation

    Visit the website to download the version that fits your platform.

    Build from source

    $ git clone <repo> mjml-app && cd mjml-app
$ npm install && npm run package

    Binary will be created in the releases folder.
